Settings Packets ​

Plugin Channel Identifier: openboatutils:settings

Contains packets that modify the current context (by default this is the default context) with the exception of the version packet, which is here due to backwards compatibility

Serverbound ​

Version ​

Sends the server the version of OpenBoatUtils we are currently running. This is sent once on join.

As of 0.5.0 an unstable flag is also sent after the first short. It is intended to block use of potentially broken development builds without having to impact the version Ids. probably reject builds of these for most people

Take care to only read the unstable flag when the version id is >= 19 (0.5.0).

Packet IDVersionUnstable (as of 0.5.0)
0 (short)Version ID (int)boolean

Set Ten Step Interpolation Non-Context ​

Sets if the boat interpolation is changed from 3 steps to 10.

Packet IDEnabled (default false)
29 (short)boolean

WARNING

This affects all boats and is handled completely separately to contexts, a reset packet won't clear this setting but relogging will.

Why do this?

Before 1.21.3 boats interpolated over 10 ticks, after this point it was changed to 3 ticks by mojang. Most servers will set this to true upon join to replicate the pre 1.21.3 behaviour which is usually essential for racing.

Set Collision Resolution ​

Sets the amount of times the boat's movement step is divided, to try to avoid incorrect corner collisions at higher speeds.

This is capped to 50 but a value of like 5 is usually more than enough.
